用Hexo创建博客 (windows版)

（一）安装一些必要的环境

  • Node.js
  • Git

Git 配置 user 和 email

安装淘宝源，速度快，但非必要

（二）博客初始化、基础操作

1：安装hexo框架

==↑以上步骤只需要弄一遍即可，不需要重复弄==

2：初始化博客框架

cmd进入博客项目文件夹，如果哪一步出问题了，直接把该文件夹删除，重新来一遍

3：启动博客 (查看本地是否能正常运行)

4：部署博客

5：添加新博客文章 并 更新博客

解决每次部署都需要输入github账号和令牌的问题

（三）博客个性化

去hexo官网找主题，点开跟随步骤安装
